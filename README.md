# Skyward Raid 2077

Skyward Raid 2077 是一个原创 HTML5 Canvas 纵版飞行射击游戏项目。玩家驾驶未来战机在高空战场中穿梭，自动开火、闪避弹幕、收集强化，并在不断升级的敌潮和 Boss 战中推进关卡。

## 项目亮点

- 单文件即可运行，所有核心玩法都集中在 `空中突袭.html`
- 原创未来空战主题，包含关卡推进、无尽模式、战机选择和成长反馈
- 内置 Canvas 渲染、输入控制、音效、界面状态和本地设置保存
- 支持鼠标与触屏操作，适合桌面和移动浏览器快速试玩

## 运行方式

直接用浏览器打开 `空中突袭.html`。

也可以在项目目录启动一个静态服务器后访问：

```powershell
python -m http.server 8000
```

然后打开 `http://localhost:8000/空中突袭.html`。
